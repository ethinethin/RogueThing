# RogueThing
Rogue-like thing

# Compile
To compile and run:

    git clone https://github.com/ethinethin/RogueThing
    cd RogueThing
    make
    ./nrl

You may need to install ncurses - see: https://ftp.gnu.org/gnu/ncurses/

# Controls

Move the character:

WASD - move up/left/down/right

QEZC - move diagonally

S - wait

L - enter look mode

P - quit
